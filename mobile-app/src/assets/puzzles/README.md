# Puzzle Images

Place your puzzle images here for the Sliding Puzzle game.

## Requirements:

- **Format**: JPG or PNG
- **Size**: At least 600x600 pixels (square images work best)
- **Names**:
  - `landscape1.jpg`
  - `landscape2.jpg`
  - `landscape3.jpg`

## Suggested Images:

1. **landscape1.jpg** - Mountain Lake scene
2. **landscape2.jpg** - Sunset Beach scene
3. **landscape3.jpg** - Forest Path scene

You can use any images you want! Nature landscapes, cityscapes, artwork, animals, etc.

## Where to Find Free Images:

- [Unsplash](https://unsplash.com/) - Free high-quality photos
- [Pexels](https://www.pexels.com/) - Free stock photos
- [Pixabay](https://pixabay.com/) - Free images

## Quick Setup:

```bash
# Download 3 square images and place them here as:
# - landscape1.jpg
# - landscape2.jpg
# - landscape3.jpg
```

The game will automatically use these images!
